ТЕМА №1 "Знакомство с GIT"
Заходим в аккаунт GIT -> В верхнем меню выбираем категорию "Repositories" -> В правом верхнем углу находим зеленую кнопку с надписью "New" ->  В графе "Repository name" пишем названия своего репозитория -> В графе "Choose visibility" выбираем видимость своего репозитория для других пользователоей -> Нажимаем зеленую кнопку "Create repository" для создания репозитория -> для загрузки файла можно просто его перетащить с рабочего стола
